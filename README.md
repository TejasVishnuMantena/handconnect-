# 🚀 Advanced AR Hand Tracking & X-Ray Vision Engine

A futuristic browser-based **Augmented Reality Hand Tracking System** built using **HTML, JavaScript, Canvas API, and MediaPipe**.  
This project combines:

- ✨ Neon AR hand tracking
- ⚡ Real-time gesture detection
- 🌌 Cyberpunk visual effects
- 🧠 AI-style X-Ray portal rendering
- 🎵 Interactive audio feedback
- 🔥 Particle physics engine
- 🖐️ Multi-hand tracking
- 🎨 Dynamic themes and animations

---

# 📌 Project Overview

This repository contains two immersive web experiences:

## 1️⃣ Neon Aura AR (`connecthand.html`)

An advanced hand-tracking visualizer with:

- Real-time hand landmark tracking
- Neon glowing fingertips
- Gesture detection
- Shockwave and ripple effects
- Cyberpunk themes
- Interactive audio engine
- Matrix rain background
- Finger lightning connections
- Particle effects

---

## 2️⃣ X-Ray Vision Portal Engine (`xrayvision.html`)

A futuristic sci-fi body visualization system featuring:

- MediaPipe Holistic tracking
- X-ray style body rendering
- Neural particle system
- Portal interaction mechanics
- HUD overlays
- Body skeleton visualization
- Glitch effects
- Scanline animations
- Energy streams and sparks
